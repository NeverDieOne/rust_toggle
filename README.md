# Rust Toggle Track Wrapper

CLI обёртка для Toggle Track

## Использование

- Получение справки
```bash
tgl --help
```

- Авторизация
```bash
tgl auth TOGGLE_TOKEN
```

- Запуск отсчёта
```bash
tgl start "Some description"
```

- Остановка отсчёта
```bash
tgl stop
```
